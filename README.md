# Website Pencari Kerja Disnaker Kabupaten Cirebon

Proyek ini merupakan **tugas akademik (Tugas Akhir / Skripsi)** yang dikembangkan sebagai bagian dari studi pada Program Studi Teknik Informatika.  
Penelitian dan pengembangan sistem dilakukan dengan **studi kasus pada Dinas Ketenagakerjaan (Disnaker) Kabupaten Cirebon**.

Website ini bertujuan untuk mendukung **digitalisasi proses rekrutmen tenaga kerja**, dengan menyediakan platform terintegrasi yang menghubungkan **pencari kerja**, **perusahaan (recruiter)**, dan **admin Disnaker** dalam satu sistem berbasis web.
